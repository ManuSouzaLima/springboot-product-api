# Sistema de cadastro de itens com Spring Boot
## Sobre o Projeto

API REST para gerenciamento de estoque, permitindo realizar operações de cadastro, atualização, listagem e remoção de produtos através de operações CRUD integradas ao banco de dados. O objetivo da aplicação é facilitar o controle e a administração dos itens em estoque de forma simples e eficiente.
